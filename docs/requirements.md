# 万能视频下载网站 - 需求分析文档

> 版本：V1.0  
> 创建日期：2026-05-19  
> 状态：已确认

---

## 一、项目背景

当前市场上存在大量下载保存视频到本地的需求，但主流平台（B站、抖音、视频号等）普遍存在以下痛点：

- **无下载入口**：大多数平台不提供视频下载功能
- **下载不便**：不支持批量下载、有次数/速度限制
- **清晰度受限**：即使能下载，清晰度往往不够
- **跨平台困难**：不同平台需要不同的工具，操作繁琐
- **移动端缺失**：大多数下载工具只支持桌面端

因此，需要开发一个**万能视频下载网站**，提供统一的、跨平台的视频下载体验。

## 二、目标用户

| 用户群体 | 核心需求 | 使用场景 |
|---------|---------|---------|
| 内容创作者 | 下载素材视频用于二次创作 | 桌面端为主 |
| 普通用户 | 保存喜欢的视频到本地离线观看 | 手机端为主 |
| 教育工作者 | 下载教学视频用于课堂播放 | 桌面端为主 |
| 研究人员 | 收集视频数据用于分析 | 桌面端为主 |

## 三、功能需求

### 3.1 V1 核心功能（本次实现）

#### F1: 视频链接解析
- **描述**：用户粘贴视频 URL，系统自动识别平台并解析视频信息
- **输入**：视频 URL（支持多种平台格式）
- **输出**：视频标题、封面、作者、时长、可用格式/清晰度列表
- **优先级**：P0（最高）

#### F2: 视频下载
- **描述**：用户选择清晰度和格式后，一键下载视频到本地
- **支持格式**：MP4（视频）、MP3（仅音频）、WebM
- **支持清晰度**：取决于源平台，最高支持 4K
- **优先级**：P0（最高）

#### F3: 多平台支持
- **描述**：支持主流视频平台的视频下载
- **已确认支持的平台**（基于 yt-dlp 2026.03 版本）：

| 平台 | 支持状态 | 备注 |
|------|---------|------|
| B站（Bilibili） | 完整支持 | 含番剧、音频、收藏夹、播放列表 |
| 抖音（Douyin） | 支持 | 可能需要 cookies |
| YouTube | 完整支持 | 含播放列表、字幕、直播回放 |
| TikTok | 支持 | - |
| 小红书 | 支持 | - |
| Twitter/X | 支持 | - |
| Instagram | 支持 | - |
| Facebook | 支持 | - |
| 爱奇艺 | 支持 | - |
| 优酷 | 支持 | - |
| 其他 | 1800+ 网站 | 通过 yt-dlp 生态支持 |

- **暂不支持**：微信视频号、快手（后续可扩展自定义 extractor）
- **优先级**：P0（最高）

#### F4: 移动端适配
- **描述**：网站在手机浏览器上同样能流畅使用
- **要求**：响应式布局，触控友好，下载链接可直接唤起手机保存
- **优先级**：P0（最高）

#### F5: 精美独特的 UI 界面
- **描述**：避免千篇一律的蓝紫 AI 风格，打造暗色高级感设计
- **设计参考**：ai.codefather.cn 的布局结构（非配色）
- **配色方向**：暗色背景 + 暖色渐变强调色（珊瑚橙→琥珀金）
- **优先级**：P0（最高）

## 四、非功能需求

### 4.1 性能要求
- 视频解析响应时间：< 5 秒
- 页面首屏加载时间：< 2 秒
- 支持并发用户数：初期 10+（本地开发阶段）

### 4.2 兼容性要求
- 浏览器：Chrome、Firefox、Safari、Edge 最新版本
- 移动端：iOS Safari、Android Chrome
- 屏幕尺寸：375px ~ 1920px 全覆盖

### 4.3 架构约束
- **无数据库**：V1 不使用任何数据库，保持轻量
- **无用户系统**：V1 无需注册登录
- **合规性**：使用开源项目 yt-dlp（MIT 协议），合规可商用

## 五、技术约束与依赖

| 依赖项 | 版本要求 | 说明 |
|--------|---------|------|
| Python | 3.11+ | 后端运行环境 |
| Node.js | 18+ | 前端构建环境 |

## 六、风险评估

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| 平台反爬策略升级 | 部分平台下载失败 | 及时更新 yt-dlp 版本 |
| 视频文件过大 | 服务器存储/带宽压力 | 流式传输，及时清理临时文件 |
| 版权风险 | 法律合规问题 | 仅提供工具能力，由用户对内容负责；添加免责声明 |
| yt-dlp 更新频繁 | API 可能变动 | 封装抽象层，隔离变化 |
